

#### 介绍
向量整数除法指令
除法与余数指令等同于RISC-V标准标量整数乘除法，对于极端输入，结果相同。
DIV 和 DIVU 指令执行 XLEN 位有符号和无符号整数除法，将 rs1 除以 rs2（向零取整）。REM 和 REMU 指令提供相应除法运算的余数。对于 REM 指令，非零结果的符号与被除数（dividend）的符号相同。

#### 注意
对于有符号和无符号除法，除溢出情况外，均满足：被除数=除数×商+余数。
若需从同一次除法运算中同时获得商和余数，推荐的代码序列为：DIV[U] rdq, rs1, rs2；REM[U] rdr, rs1, rs2（rdq 不能与 rs1 或 rs2 相同）。微架构可将这两条指令融合为一次除法操作，而非执行两次独立的除法。
DIVW 和 DIVUW 是 RV64 指令，将 rs1 的低 32 位作为有符号或无符号整数除以 rs2 的低 32 位，将 32 位商存入 rd，并符号扩展至 64 位。REMW 和 REMUW 是 RV64 指令，提供相应的有符号和无符号取余运算。REMW 和 REMUW 在所有情况（包括除零）下均将 32 位结果符号扩展至 64 位。
除零和除法溢出的语义总结见表 1。除零的商设置为全 1（所有位），除零的余数等于被除数。有符号除法溢出仅发生在用 -1 除最小负整数时。有符号除法溢出的商等于被除数，余数为零。无符号除法不会发生溢出。
